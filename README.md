## 주요 기능

### 쇼핑몰
- 쇼핑몰메인 : 상품 클릭 시 상세 페이지 이동 및 장바구니 담기 기능 
- 장바구니 : 상세 페이지에서 담은 물품들 리스트 출력 및 결제 페이지 이동 (로그인 상태가 아니라면 로그인 페이지로 이동)
- 결제 : 물품을 수령할 고객의 전화번호 및 주소를 입력 및 I`mport 결제 API를 사용
